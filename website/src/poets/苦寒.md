# 《苦寒》

谁在语，

　　秋空半夜水净明，淤塞恬湖蛮。

　　走地笋叶风忽散，亭亭玉荷开。

语天阙，

　　珠璇穹星渐掩埋，半朵桃花裁。

　　落红残忆皆作古，把酒再言欢。

　　                                        ` ——2023.4.26`
